# L.Corp Dub Mod // 脑叶公司配音Mod

## 关于

众所周知，脑叶全部剧情是没有配音的，然而图书馆以及巴士都是有配音的，为了弥补这种缺憾，于是有了此项目

我只会对Angela的剧情对白进行配音补全，但是只要提供的音频文件名和Mod需要的相同，即使是其他Sephirah说话的对白也可以配上音频。不过我不会做这些工作。

目前进度：Day1-Day3

## 使用

本项目依赖MelonLoader加载（因为本人不想去学Harmony啥的），所以可能和传统Mod文件冲突，但是代码是开源的，内容也很简单，有能力的随便适配一下PR一下就好了。

下载地址：[百度网盘](https://pan.baidu.com/s/51ltesOGWNvaHiW-tH1ZQeg )

下载后把解压包的内容拖至脑叶公司游戏的根目录下即可，然后启动游戏就可以用啦！（如图）
